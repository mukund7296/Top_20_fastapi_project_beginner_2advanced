# 🚀 FastAPI Learning Journey (Beginner to Advanced)

This repository is a structured collection of **FastAPI projects**, designed to take you from **basic concepts to advanced, production-level applications** through hands-on practice.

The goal is simple: **learn FastAPI by building real projects step by step**.

---

## 📚 What You’ll Learn

* Building REST APIs with FastAPI
* Request handling (path, query, body)
* Data validation using Pydantic
* Authentication (JWT, OAuth2)
* Database integration (SQLAlchemy, PostgreSQL)
* Async programming & WebSockets
* Background tasks and file handling
* Scalable architecture and API design
* Production-ready backend development

---

## 🗺️ Learning Path

### 🟢 Beginner

Start with FastAPI fundamentals:

* Basic routes and responses
* CRUD operations
* Request validation
* Error handling
* Working with external APIs
  
🟢 Beginner (Days 1–5)

#### 1. Hello Health API — Basic routes, request/response, status codes
#### 2. Personal Todo API — CRUD with in-memory storage
#### 3. Student Grade Manager — Pydantic models, data validation
#### 4. Currency Converter API — Query params, external API calls
#### 5. Weather Info API — Path params, error handling, HTTPException


---

### 🟡 Intermediate

Move to real-world backend features:

* Authentication and authorization
* Database integration
* File uploads and background tasks
* Role-based access control
* Business logic implementation
* 🟡 Intermediate (Days 6–12)

#### 6. User Auth API — JWT tokens, OAuth2, password hashing
#### 7. Blog Platform API — SQLite + SQLAlchemy, relationships
#### 8. E-Commerce Product API — PostgreSQL, filtering, pagination
#### 9. File Upload Service — File handling, background tasks
#### 10. Inventory Manager API — Role-based access control (RBAC)
#### 11. Appointment Booking API — Date/time logic, conflict validation
#### 12. Notification Service API — Background tasks, email/webhook triggers

### 🔴 Advanced

Build scalable and modern systems:

* WebSockets and real-time apps

🔴 Advanced (Days 13–17)

#### 13. Real-Time Chat API — WebSockets, async messaging
#### 14. AI Document Summarizer API — LLM integration, streaming responses
